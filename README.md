# lapormasit
Sistem Pelaporan &amp; Penanganan Kendala Teknis IT Support adalah platform untuk menerima, memantau, dan menyelesaikan laporan gangguan dari setiap unit melalui Ticket Support. Sistem ini membantu IT mendata, memproses, dan membuat laporan berkala secara terstruktur, transparan, serta terdokumentasi dengan baik.
