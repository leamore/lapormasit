# Sistem Ticketing IT Support RS
Sistem Pelaporan & Penanganan Kendala Teknis IT Support merupakan platform terintegrasi yang dirancang untuk memfasilitasi proses pelaporan, pemantauan, dan penyelesaian kendala teknis di lingkungan kerja secara terstruktur dan terdokumentasi.

Melalui sistem ini, setiap unit dapat mengajukan laporan kendala teknis melalui mekanisme Ticket Support dengan mengisi informasi terkait unit dan deskripsi permasalahan. Setiap laporan yang masuk akan tercatat secara otomatis dalam dashboard IT Support untuk diproses sesuai dengan tingkat prioritas dan kategori permasalahan.

- Di sisi IT Support, sistem menyediakan fitur monitoring yang memungkinkan petugas untuk:
- Menerima dan mengelola tiket masuk
- Melakukan pencatatan tindak lanjut dan status penyelesaian
- Melacak histori penanganan kendala
- Memonitor IP address dan perangkat pelapor (jika diaktifkan)
- Menghasilkan laporan berbasis periode (harian, mingguan, bulanan, dan tahunan)

Selain itu, sistem dilengkapi dengan fitur keamanan seperti penggunaan kode akses untuk pengajuan tiket guna meminimalkan potensi spam atau laporan tidak valid. Dengan adanya sistem ini, proses penanganan kendala teknis menjadi lebih transparan, terdokumentasi, terukur, serta dapat dijadikan dasar evaluasi kinerja IT Support secara berkala.
